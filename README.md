# Odoo 15 Hospital Management Module

A custom Odoo 15 module built as a practice project 
following the OdooMates tutorial.

## Features
- Patient Management
- Doctor Management
- Appointment Booking with Workflow
- PDF Report Generation

## Tech Stack
- Python 3 (Odoo ORM)
- XML (Views)
- CSS (Styling)
- PostgreSQL (via Docker)

## Setup
1. Clone this repo into your Odoo addons folder
2. Restart Odoo and update apps list
3. Install the module from the Apps menu
